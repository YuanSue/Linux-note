DB2開啟問題

正常開啟VM會帶DS起來，若無法自動帶起，且用手動開啟，還是無法正常啟動，則有很大的機率是DB2沒有正常啟動
開啟WAS步驟:
1. 先手動讓DSEngine啟動 ⮞ 2. 切到db2inst1: su - db2inst1讓db2啟動: db2start 如果成功會看到：SQL1063N DB2START processing was successful. ⮞
3. 確認 Database Manager 是否跑起來: db2 get dbm cfg | head -5 ⮞ 4. 確認進程: ps -ef | grep db2sysc ⮞
5. 列出 active databases: db2 list active databases(如果成功會看到： SQL1611W No data was returned by Database System Monitor.) ⮞
6.restart: cd /u01/IBM/InformationServer/ASBServer/bin/ ⮞ ./MetadataServer.sh restart
